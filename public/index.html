🔑 API トークン

スクリプトや自動化からの API アクセスに使う固定トークンです。再生成すると古いトークンは即座に無効になります（ブラウザのログインセッションには影響しません）。

🛡️ 脅威検出設定

外部の脅威インテリジェンスフィードからマルウェアC2/配布元IPリストをダウンロードし、NAT セッションと突合します。

同一宛先への一定間隔の通信（C2ハートビートの典型パターン）を自動検出します。候補は通信ログタブのバナーに表示されます。

既知の良性 telemetry（IoT ベンダーのクラウド等）を候補から除外します。

ホスト名が解決できない宛先にも効きます。脅威フィードに該当する宛先は除外されず候補に残ります。

📡 Data Sources

Configure the data sources used to resolve destination hostnames.
Domain names from enabled sources appear in the connection log and related views.

Tail the dnsmasq query log on the server and attach per-device DNS resolution results to destination hostnames.
These names take priority over PTR reverse lookups, so meaningful names such as example.com are shown.

Parse Yamaha syslog [INSPECT] entries in real time to supplement short-lived TCP sessions missed by 60-second polling.
Specify the Yamaha syslog file path received by the server.

Parse Yamaha syslog [DHCPD] entries (Allocates/Extends) in real time and maintain IP-to-MAC mappings.
This improves source-device identification. Usually this uses the same path as [INSPECT].
